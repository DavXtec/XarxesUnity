Socket connection and movement example context:

	- Open MovExSampleScene
	- Use connect button to try and connect to Node.JS Server 
	- After connecting, server will show simulations on all connected players (except self), you can test this by making a build and running multiple instances at the same time.
	- You may use WASD to move around and Escape to locally Pause and handle disconnections
	- Unexpected shutdown of the server should be properly handled from Unity
